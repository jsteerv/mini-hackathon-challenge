# Create Archon PRP

## Request: $ARGUMENTS

Use agent-organizer to create a comprehensive PRP for: $ARGUMENTS

The agent-organizer will:
1. Intelligently analyze the request to determine if PRP methodology is appropriate
2. If a PRP is needed, orchestrate the research and creation process
3. Use Archon for project and task management
4. Assign appropriate agents based on the specific requirements

The agent-organizer has knowledge of all available agents and the PRP methodology. It will make intelligent decisions about which agents to use based on the nature of the request.

If a project_id is provided, it will add the PRP to the existing project. Otherwise, it will create a new Archon project.
